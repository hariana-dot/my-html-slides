# HBM Competition 2026 — SK hynix vs Samsung vs Micron

Professional visual presentation comparing the three leading HBM (High Bandwidth Memory) manufacturers in the context of the ongoing AI memory supercycle.

## Features
- **Visual Heavy Mode** with custom AI-generated backgrounds
- Strong investment thesis angle (why HBM is structurally different from traditional DRAM/NAND)
- HBM4 technology deep-dive and head-to-head comparison
- Includes China competitive landscape (CXMT and others)
- Clean, modern Reveal.js design optimized for presentation or GitHub Pages

## Live Preview
Open `index.html` in any modern browser.

## GitHub Pages Deployment
1. Push this folder to `hariana-dot/my-html-slides/hbm-competition-2026`
2. Enable GitHub Pages on the repository (Settings → Pages → Deploy from branch → main / root)
3. Access at: `https://hariana-dot.github.io/my-html-slides/hbm-competition-2026/`
